# 🎯 ResumeAI — Smart Resume Analysis & ATS Optimization

ResumeAI is a professional, AI-driven web application designed to help job seekers outrank the competition. By performing a deep semantic analysis of your resume against a specific job description, ResumeAI identifies critical skill gaps, calculates alignment scores, and provides custom interview preparation questions.

![ResumeAI Preview](https://via.placeholder.com/800x400?text=ResumeAI+Interface+Preview)

## 🚀 Features

- **Smart Resume Parsing**: Extracts text and entities from PDF resumes with high accuracy.
- **ATS Alignment Scoring**: Calculates an overall match percentage based on weighted skill overlap (80%) and general context (20%).
- **Skill Gap Identification**: Visualizes "Strengths" (matched skills) and "Gaps" (missing skills) using clean, color-coded UI chips.
- **AI-Powered Interview Prep**: Dynamically generates practice questions based on your specific profile and the target job description.
- **Actionable Insights**: Provides specific suggestions to improve resume content and structure (e.g., missing Experience or Education sections).
- **Responsive Professional UI**: Built with a modern, high-converting aesthetic inspired by industry leaders.

## 🛠️ Tech Stack

- **Backend**: Python, Flask
- **File Processing**: PyPDF2
- **Frontend**: Tailwind CSS, FontAwesome, JavaScript
- **Deployment Ready**: Lightweight dependencies for easy setup on Windows/Linux/macOS.

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Salman-Dev262/AI-Resume-Analyzer.git
   cd AI-Resume-Analyzer
   ```

2. **Install dependencies**:
   Ensure you have Python installed, then run:
   ```bash
   py -m pip install -r requirements.txt
   ```

## 🏃 How to Run

1. **Start the Flask server**:
   ```bash
   py app.py
   ```

2. **Access the application**:
   Open your browser and navigate to:
   `http://127.0.0.1:5000/`

## 📖 Usage

1. **Home**: Explore the features and process guide.
2. **Analyze**: Scroll to the analyzer section, upload your resume (PDF), and paste the job description.
3. **Report**: View your deep-dive match report, practice your STAR-method interview answers, and implement the suggested improvements.
